# Youtube Clone
Application that is a clone of youtube and function just the same.



## Dev Dependencies
- Material UI
- React
- React DOM
- React Router DOM

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
